# Community Code of Conduct

Daily Berlin Jobs is built for the public and should be a welcoming place to
contribute.

## Our standards

Be respectful, assume good intent, give actionable feedback, and make room for
people with different backgrounds and experience levels. Disagreement about
code or product direction is welcome; harassment, discrimination, personal
attacks, intimidation, and publishing another person's private information are
not.

## Enforcement

Project maintainers may edit or remove comments, commits, issues, and other
contributions that violate these standards. Temporary or permanent bans may be
used when necessary to protect the community.

Report conduct concerns privately to the repository owner through their GitHub
profile. Do not open a public issue containing personal or sensitive details.
